# Account Health Audit Checklist — ロブロックス海外

## 実施頻度: 月1回（毎月1日推奨）

---

## 1. ブラウザ分離チェック

- [ ] Firefox（グローバル専用）でのみロブロックス海外アカウントにログインしている
- [ ] Chrome（日本用）でグローバルアカウントにログインしていない
- [ ] Firefoxの履歴に日本語サイトの閲覧がない
- [ ] Firefoxのブックマークに日本語サイトがない

## 2. アカウント設定チェック

- [ ] Google Account Language: English (US)
- [ ] YouTube Location: United States
- [ ] YouTubeの表示言語が英語になっている
- [ ] おすすめフィードに日本語コンテンツが表示されていない

## 3. 視聴履歴チェック

- [ ] YouTube視聴履歴を確認 → 日本語動画の視聴がない
- [ ] YouTube検索履歴を確認 → 日本語での検索がない
- [ ] 英語コンテンツ（Roblox, Minecraft, Skibidi等）の視聴が直近1ヶ月で10件以上ある
  - （アルゴリズムを英語圏にトレーニングするため）

## 4. YouTube Studio アナリティクスチェック

- [ ] 視聴者の上位国を確認 → 日本が上位3位以内に入っていない
  - もし日本が上位に来ている場合: アルゴリズム汚染の可能性 → ブラウザ履歴を再確認
- [ ] 視聴者の年齢層を確認 → 13-24歳が主要層である
- [ ] トラフィックソースを確認 → 「おすすめ」からの流入がある
  - おすすめがない場合: メタデータ/タグの見直しが必要

## 5. メタデータ品質チェック

- [ ] 直近投稿の全タイトルが英語のみ（日本語混入なし）
- [ ] 直近投稿の全説明文が英語のみ
- [ ] 直近投稿の全タグが英語のみ
- [ ] 説明文にAI開示が含まれている
- [ ] 説明文に人間クレジット（Sound Design / Voice / Story）が含まれている

## 6. 説明文個別化チェック

- [ ] 直近5本の説明文を比較 → 全て異なるストーリーサマリーを持つ
- [ ] テンプレート部分（AI開示、クレジット）以外の文面がそれぞれユニーク
- [ ] `validate_authenticity.py` を直近のスクリプトに実行して警告がない

## 7. コンテンツ多様性チェック

- [ ] 直近10本で同一ストーリーアーキタイプが3連続していない
- [ ] 直近10本で同一背景/マップが3連続していない
- [ ] `validate_originality.py` を直近のスクリプトに実行して警告がない

## 8. コミュニティ活動チェック

- [ ] 直近1ヶ月でコメントに返信している（最低5件）
- [ ] コミュニティ投稿を1回以上行っている（任意、推奨）
- [ ] スパムコメント/日本語コメントの管理を行っている

## 9. 制作証拠チェック

- [ ] 直近投稿の全動画に`Production_Evidence/`フォルダがある
- [ ] 各フォルダに foley_raw/, veo3_prompts.md, storyboard.md, edit_notes.md がある
- [ ] Google Driveバックアップが最新の状態

---

## 問題発見時のアクション

| 問題 | アクション |
|---|---|
| 日本語コンテンツ汚染 | Firefox履歴を全削除、英語コンテンツを50件以上視聴して再トレーニング |
| 日本からの視聴者増加 | タイトル/タグを見直し、日本語検出がないか確認 |
| 説明文テンプレ化 | Description_Guide.mdを参照し、過去と異なる文面に書き直し |
| アーキタイプ偏り | 次回はバランスを取るために別のアーキタイプを選択 |
| 制作証拠不足 | 今後の動画から必ず記録。過去分も可能な限り遡って記録 |
